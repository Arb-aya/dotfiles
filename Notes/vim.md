## vim-surround
Ctrl + S B = {}
Ctrl + S b = ()
Ctrl + S r = []
Ctrl + S " = ""

Use Ctrl + S Ctrl + S (B/b/r/"/') will put a line between them.

## Dirvish
`-` to view files
`:e` to create a new file
`% <tab>` to expand to full file path

Use `:Shdo` to modify selected file paths and `Z!` or `!%` to execute it
